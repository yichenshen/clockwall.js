clockwall.js
==================

An implementation of this awesome project: http://www.humanssince1982.com/a-million-times, in Javascript.

##Dependencies

 - [Raphael](http://raphaeljs.com/)
 - [GSAP](http://greensock.com/gsap)
  - TweenLite
  - TimelineLite
  - Raphael Plugin
